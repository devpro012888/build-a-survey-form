# Build A Survey Form
FreeCodeCamp Certification Project - Build a Survey Form

## Description
This project is a responsive survey form designed to collect feedback from FreeCodeCamp students. The form gathers personal information, learning preferences, and suggestions for improvement to help enhance the FreeCodeCamp curriculum and resources.

## Features
- **Personal Information Collection**: Captures name, email, age, and current role
- **Certification Preferences**: Multiple choice checkboxes for favorite FreeCodeCamp certifications
- **Time Commitment**: Radio buttons to indicate weekly study hours
- **Feedback Sections**: Text areas for improvement suggestions and additional comments
- **Form Validation**: Required fields for essential information
- **Responsive Design**: Works on various screen sizes

## Form Fields
### Personal Information
- First and Last Name (required)
- Email (required)
- Age (optional, 5-99 range)
- Current Role (dropdown selection)

### Favorite Certifications
- Responsive Web Design Certification
- JavaScript Certification
- Front End Development Libraries Certification
- Python Certification
- Relational Databases Certification
- Back End Development and APIs Certification
- Certified Full Stack Developer Curriculum

### Study Habits
- Hours per week spent on FreeCodeCamp (1-5, 6-10, 11-15 hours)

### Feedback
- Suggestions for improvement
- Additional comments

## Technologies Used
- HTML5
- CSS (for styling, if applicable)

## How to Use
1. Open `index.html` in a web browser
2. Fill out the survey form with your information
3. Select your favorite certifications
4. Indicate your weekly study hours
5. Provide feedback in the text areas
6. Click the Submit button to send the form

## Project Requirements Met
This project fulfills the FreeCodeCamp Responsive Web Design Certification requirements for the "Build a Survey Form" project, including:
- Proper HTML form structure
- Required fields and validation
- Various input types (text, email, number, select, checkbox, radio, textarea)
- Semantic HTML with fieldsets and legends
- Accessible form labels
